# **App Name**: AI Studio Converter

## Core Features:

- Project Upload: Allow users to upload the React project exported from Google AI Studio.
- Component Conversion: Automatically convert React components to Next.js components, adapting file structure and syntax as needed.
- API Route Creation: Generate Next.js API routes for any API endpoints used in the original React app.
- Dependency Update: Update dependencies to Next.js-compatible versions, where applicable.
- Error Reporting: Report any conversion errors or compatibility issues to the user, suggesting manual adjustments.
- Code Validation: Use AI tool to evaluate the converted code for correctness and propose necessary fixes.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and stability.
- Background color: Light gray (#F5F5F5) for a clean, neutral backdrop.
- Accent color: Teal (#4DB6AC) for interactive elements and calls to action.
- Font pairing: 'Inter' (sans-serif) for body text and 'Space Grotesk' (sans-serif) for headlines.
- Use simple, outline-style icons to represent different functions and feedback messages.
- Clean, grid-based layout with ample white space to improve readability and reduce visual clutter.
- Subtle transitions and animations to provide feedback on user interactions.
